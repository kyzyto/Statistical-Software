Accessing SAS Software and Setting up Practice Files

To complete the practices in this course, you MUST access SAS Software and set up the practice files. 

1. Choose the software you are going to use by reading this PDF: Comparing Free Software Offered by SAS. Follow the
   links to either create an account for SAS OnDemand for Academics or download SAS University Edition.   

2. After you access SAS software, follow the instructions in ONE of these PDFs: 
     **Data Setup for SAS OnDemand for Academics 
       OR 
     **Data Setup for SAS University Edition.


NOTE: You only have to do this data setup ONE TIME for all the following courses:

SAS 9.4 Cert Prep: Essentials of SAS Programming
SAS 9.4 Cert Prep: Accessing Data with SAS Programming
SAS 9.4 Cert Prep: Exploring and Validating Data with SAS Programming
SAS 9.4 Cert Prep: Preparing Data with SAS Programming
SAS 9.4 Cert Prep: Analyzing and Reporting on Data with SAS Programming
SAS 9.4 Cert Prep: Exporting Results with SAS Programming
SAS 9.4 Cert Prep: Using SQL in SAS
